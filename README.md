# Algorand Project – Voting System

##  Project Description
This project is a decentralized voting system built on the Algorand blockchain. It ensures secure, transparent, and tamper-proof elections.

## Features
- Secure voting using blockchain
- Transparent and verifiable results
- Tamper-proof data storage

## Technologies Used
- Algorand Blockchain
- Python
- PyTeal
